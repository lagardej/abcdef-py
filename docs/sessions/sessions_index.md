# Session Log Index

Master index of all working sessions. Post-mortem reference this log to track decision quality, iteration cycles, and
architectural effectiveness.

---

Each session captures:

- **Decisions Made** — Architectural, design, or implementation choices
- **Assumptions** — What we took as given without verification
- **AI Iterations** — How many rounds of clarification; major pivots
- **Blockers** — Uncertainties, blocked decisions, deferred choices
- **Artefacts** — Commits, files created, tests written
- **Lessons** — What worked, what could be better

Collected at the end of each feature for reflection.
